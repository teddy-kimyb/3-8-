🔧 Firebase 설정

최초 1회만 진행하면 됩니다
⚠️ Firebase 설정이 필요합니다
아래 순서대로 진행 후 설정값을 입력하면 모든 기기에서 데이터가 공유됩니다.

📋 설정 순서

1

Firebase 프로젝트 만들기

console.firebase.google.com 접속
→ Google 계정으로 로그인
→ "프로젝트 추가" 클릭
→ 프로젝트 이름: 3-8반상담 입력
→ Google Analytics: 사용 안 함 선택
→ "프로젝트 만들기" 클릭

2

Realtime Database 만들기

왼쪽 메뉴 "빌드" → "Realtime Database"
→ "데이터베이스 만들기" 클릭
→ 위치: 미국(기본값) 그대로
→ 보안 규칙: "테스트 모드로 시작" 선택
→ "사용 설정" 클릭

✅ 완료되면 주소창에 https://3-8반상담-default-rtdb.firebaseio.com 형태의 URL이 보입니다. 이게 databaseURL 값입니다.
3

웹 앱 등록하기

왼쪽 상단 ⚙️ 프로젝트 설정 클릭
→ 하단 "내 앱" 섹션 → </> (웹) 아이콘 클릭
→ 앱 닉네임: 상담앱 입력
→ "앱 등록" 클릭
→ 화면에 나타나는 firebaseConfig 코드를 복사

// 이런 형태의 코드가 나타납니다:
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "3-8반상담.firebaseapp.com",
  databaseURL: "https://...firebaseio.com",
  projectId: "3-8반상담",
  ...
};
4

아래에 설정값 입력

복사한 firebaseConfig의 각 값을 아래 입력란에 붙여넣으세요.

🔑 Firebase 설정값 입력

💡 입력한 설정값은 이 기기 브라우저에 저장됩니다.
다른 기기에서도 처음 접속할 때 이 과정을 한 번씩 진행해야 합니다.

더 편하게 쓰려면: 설정값을 입력 후 생성된 HTML 파일을 GitHub에 다시 업로드하면 이후엔 설정 없이 바로 사용 가능합니다.
